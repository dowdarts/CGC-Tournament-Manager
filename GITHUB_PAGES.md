# GitHub Pages URLs

This repository is deployed to GitHub Pages with the following URLs:

## Main Tournament Management System
- **URL**: https://dowdarts.github.io/CGC-Tournament-Manager/
- **Description**: Complete tournament management system with player management, group stage, knockout brackets, and live display

## Live Tournament Display
- **URL**: https://dowdarts.github.io/CGC-Tournament-Manager/tournament-display
- **Description**: Full-screen live tournament display for public viewing
- **Features**: 
  - Real-time match updates
  - Group stage standings with scrolling
  - Knockout bracket progress
  - Championship celebration animation
  - TV mode with larger text and optimized layout
  - Automatic filtering for tournaments with recent activity

## Registration Portal
- **URL**: https://dowdarts.github.io/CGC-Tournament-Manager/registration/
- **Description**: Standalone registration portal for tournament participants
- **Features**: 
  - Tournament selection and registration
  - Player information collection
  - Payment tracking
  - Automatic confirmation emails
  - Real-time tournament availability

## Usage

### For Tournament Organizers
1. Use the main system to create and manage tournaments
2. Share the registration portal URL with participants
3. Monitor registrations in real-time
4. Use the live display for tournament presentation

### For Participants
1. Visit the registration portal URL
2. Select your tournament
3. Fill in your details
4. Complete registration
5. Receive confirmation email

## Automatic Updates

Both URLs are automatically updated when changes are pushed to the main branch through GitHub Actions.
