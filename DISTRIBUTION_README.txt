CGC Tournament Manager - Installation Instructions

IMPORTANT: You must have internet access to use this application.
This app connects to online services (Supabase) for data storage.

═══════════════════════════════════════════════════════════════

INSTALLATION:
1. Double-click "CGC Tournament Manager-Setup-1.0.0.exe"
2. Follow the installation wizard
3. Choose installation location (default is recommended)
4. Click "Install"
5. Launch from Desktop shortcut or Start Menu

FIRST RUN:
1. The app will open to the sign-in page
2. Create a new account or sign in with existing credentials
3. Your tournaments are stored securely in the cloud
4. You can access them from any computer by signing in

SYSTEM REQUIREMENTS:
- Windows 10 or later (64-bit)
- Active internet connection (required)
- 500 MB free disk space
- 4 GB RAM minimum (8 GB recommended)

FEATURES:
✓ Create and manage tournaments
✓ Group stage with round-robin play
✓ Knockout bracket generation
✓ Live scoring and standings
✓ Public display pages for fans
✓ Email notifications for players
✓ Board call system
✓ DartConnect scraping integration
✓ Professional tournament management

UNINSTALL:
Windows Settings → Apps → CGC Tournament Manager → Uninstall

TROUBLESHOOTING:
- If the app won't start, try running as Administrator
- Check your internet connection
- Make sure Windows Defender is not blocking the app
- Restart your computer after installation

DATA & PRIVACY:
- All tournament data is stored securely in the cloud
- Each user has a private account
- No local data storage
- Your data is accessible from any computer
